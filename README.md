## Overview

Modules and Functions for the mse Package

## Installation
To install this package, start R and enter:

```
install.packages("msemodules", repos="http://flr.r-universe.net/")
```

or directly from the github repository by using:

```
remotes::install_github("flr/msemodules")
```
